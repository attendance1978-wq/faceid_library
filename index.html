FaceID Library Documentation

Welcome to FaceID - Advanced Face Recognition Library for Python

Installation

pip install faceid-lib

Quick Start

from faceid import FaceID

# Initialize
face_system = FaceID()

# Register a face
face_system.register_face("person.jpg", "person", "Person Name")

# Recognize faces
results = face_system.identify_face("test.jpg")
